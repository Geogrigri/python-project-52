## Task Manager

Task Manager is a Django web application for managing tasks.

### Local development

```bash
make install
make migrate
make dev
```

### Production

Render build command:

```bash
make build
```

Render start command:

```bash
make render-start
```

Deployed application: https://python-project-52-obym.onrender.com
